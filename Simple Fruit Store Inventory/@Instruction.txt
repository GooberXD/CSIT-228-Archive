PROBLEM

Create a Java program that manages a simple fruit store inventory. 
You must create only ONE class named Fruit. In the main() method, create an array of 
Fruit objects. The program should allow the user to:

Add a fruit
Display all fruits
Exit
For each fruit, the program must display:

Fruit name
Price per kilo
Quantity in stock (kg)
Class Requirements

Class Name:

Fruit
Attributes:

name (String)
pricePerKilo (double)
quantity (double)
Constructor:

Accept name, pricePerKilo, and quantity
Required Methods:

displayFruit()

Displays name, price per kilo, and quantity (both formatted to 2 decimal places)
getTotalValue()

Returns pricePerKilo * quantity as a double
This is the total stock value of that fruit
Additional Requirements

Maximum of 5 fruits (use a plain array, no ArrayList)
Use a do-while loop for the main menu
Use switch-case for option handling
